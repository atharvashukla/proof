A Laptop with Eight Servers

The physical realization of this is to build a general-purpose laptop and operating system with eight "always on" personal services:

  1. Media/File Server
  2. Email/Messaging Server
  3. Blockchain/Decentralization Server
  4. AI-Training/Inference Server
  5. Version-Control/Editor Server
  6. Game/Simulation Server
  7. Secrets/Vault Server
  8. Web/Database Server

All accelerated down to RISC-V hardware, with a focus on rigor that reflects the name of this program.

Initially, the laptops will be able to run with their lids closed; but in the future, one could imagine composing them together into a single system image - stretching the idea to a personal data-center.

Vision

I want the Internet, once again, to be truly peer to peer; I want the personal computer, once again, to be a bicycle of the mind - one that enables Ownership, Creativity, Freedom, and Sovereignty; One that is actually personal and expressive, like the candid exchange with friends in your drawing room.
